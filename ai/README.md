# AI Logic & Experiments

This directory houses all AI-related logic, ensuring independent testing and iteration.

## Folder Purpose
- `/prompts`: Store all system prompts and templates here. Version control your prompts!
- `/experiments`: Jupyter notebooks or scripts for testing models.
- `/evaluation`: Logic to score/evaluate model outputs.
- `/vision`: Computer vision specific scripts.
- `/text`: LLM text generation logic.

## Usage
1. Add API keys to `/ai/.env`
2. Run experiments in `/experiments`
3. Export stable logic to `/ai/ai_config.py` for integration.
